## Backup

该服务是一个简单的备份服务，负责定期备份应用程序的数据文件到指定的备份目录。它支持增量备份和全量备份两种模式，并且可以配置备份的频率和保留的备份版本数量。

### 快速开始

```bash
go mod tidy
sudo go run main.go
```

注意需要 sudo 权限，因为备份目录可能需要更高的权限访问，即对 binlog 目录的读权限，也可以配置用户权限，不过这样更简单一些。

### 功能

- 每周日晚上 0 点进行全量备份
- 其余天的晚上 0 点进行增量备份，会拷贝 binlog 目录下自上次备份以来新增或修改的文件
- 如果消息丢失可采用 full + inc 的方式进行恢复
